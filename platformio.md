มาเริ่มกันเลย
## ขั้นตอนการติดตั้ง
1. เราต้อง donwload visual studio code ก่อน จากลิ้งนี้เลยครับ https://code.visualstudio.com/download
2. ให้เราเข้าโปรแกรม visual studio code แล้วคลิกที่ Extentions
3. ค้นหาว่า platformio แล้วกด install
